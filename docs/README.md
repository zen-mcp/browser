# browser-mcp documentation site

Built with [Astro](https://astro.build/) and [Starlight](https://starlight.astro.build/). Source pages are Markdown under `src/content/docs/`.

## Commands

From this directory:

| Command | Action |
| :-- | :-- |
| `yarn install --frozen-lockfile` | Install dependencies |
| `yarn dev` | Dev server (default `localhost:4321`) |
| `yarn build` | Production build to `dist/` |
| `yarn preview` | Preview the production build |

## GitHub Pages

Configured for **Project Pages**: `https://zenkiet.github.io/browser-mcp/`.  
`site` and `base` are set in `astro.config.mjs` - update if the repo name or Pages URL changes.
